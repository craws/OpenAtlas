The OpenAtlas API (Application programming interface) output is based on JSON-LD syntax and GeoJSON, and therefore could be rendered in many web mapping applications. In addition, Linked Places was used as standard.

The API links at entity views are only visible for user who have activated Show API links in their Profile.

The base url of the current OpenAtlas API is api/0.2/
