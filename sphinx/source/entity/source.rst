Source
======

.. toctree::

Written sources like for example medieval charters are mapped in OpenAtlas as a combination of
several entities.

.. include:: navigation.rst

Form fields
-----------
* :doc:`/form/name`
* :doc:`type`
* :doc:`/entity/artifact` - if the artifact contains the source
* **Content** - a source content usually contains a summary or the whole text of the source

Can be linked via tabs to
-------------------------
* :doc:`event`, :doc:`actor`, :doc:`artifact`, :doc:`place`, and subunits it references
* :doc:`reference` - where it is documented in, e.g. in various editions of charters or secondary sources
* **Translation** - respectively to the text in the original language (e.g. Latin), can be a comment too
* :doc:`file`
